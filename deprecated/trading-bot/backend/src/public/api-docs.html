AI Trading Bot Platform API

A powerful backend service for authentication, content management, and trading bot configuration.

Interactive API Documentation

About This API

The AI Trading Bot Platform API provides a comprehensive set of endpoints for building, managing, and interacting with algorithmic trading bots. It features a robust authentication system, a flexible content management system (CMS), and specialized endpoints for trading bot configuration and performance monitoring.

Note: This is an API server, not a web interface. It is designed to be consumed by frontend applications and other services. For the user interface, please use the React frontend application.

The following endpoints are available in the API. For detailed information about request and response formats, authentication requirements, and examples, please refer to the interactive Swagger documentation.
